# sign-language-live-chat
한림대학교 2022년도 1학기 캡스톤 디자인 Web part

## 개발 현황
 - 1:1 실시간 화상 채팅
 - 실시간 화상 저장

## 개발 목표
 - 1:N 화상 회의 구현
 - Text 채팅창 구현
 - TTS 데이터 표시 화면 구현
